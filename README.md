# Task Manager App 

A Python command-line task manager that evolves into a full CRUD application. Designed to simulate real-world productivity tools while practicing core programming skills.

---

##  Features
- Add, view, and delete tasks
- Mark tasks as complete or pending
- Priority and due date support (advanced version)
- Persistent storage using JSON files
- Menu-driven CLI interface

---

##  Project Stages
- v1: Basic task list  
- v2: Task status system  
- v3: File storage (JSON)  
- v4: Advanced features (priority, due dates)  
- v5: Full CLI task manager  

---

##  Technologies
- Python 3
- JSON file handling
- Data structures (lists & dictionaries)

---

##  Future Improvements
- Web app using Flask
- Database integration (SQLite)
- User accounts and authentication

---

##  Purpose
This project demonstrates CRUD operations, structured program design, and persistence in Python applications.
